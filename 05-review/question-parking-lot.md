# 问题停车场

用于保存值得记录、但不应打断当前课程顺序的问题。

## 使用规则

每个问题至少记录：

- 问题；
- 来源或触发场景；
- 与课程哪一讲相关；
- 为什么暂不展开；
- 重新进入讨论的条件；
- 当前简要判断。

## 模板

### Q-XXX：问题标题

- 日期：
- 来源：
- 课程关联：
- 分类：必要前置 / 后续主题 / 研究旁支 / 产业背景 / 弱相关
- 暂存原因：
- 重新进入条件：

**问题描述**

**当前简要判断**

**需要补充的证据或实验**

## 初始旁支

### Q-001：Diffusion Policy、ACT 与基础 BC 的关系

- 课程关联：Imitation Learning、Generative Models、Sequence Models
- 分类：后续主题
- 暂存原因：需要先掌握 BC、分布偏移、概率策略和序列策略基础
- 重新进入条件：完成 L05，并能独立解释多峰动作分布

### Q-002：VLA 在工业机械臂中的合理边界

- 课程关联：Visuomotor Policy、Transfer Learning、Multi-Task Learning、Open-World Robotics
- 分类：研究旁支
- 暂存原因：需要先建立控制、规划、感知和策略学习的统一比较框架
- 重新进入条件：完成 Perception 模块并建立至少一个传统控制对照实验

### Q-003：Residual RL 用于接触与装配

- 课程关联：Robotic Control、Actor-Critic、Model-Based Decision Making
- 分类：后续实验主题
- 暂存原因：需要先完成传统控制基线和基本连续控制 RL
- 重新进入条件：完成 L09–L12，并在 ManiSkill 中建立接触任务

### Q-004：Sim2Real 是否把瓶颈转移到仿真搭建

- 课程关联：Dynamics Learning、Simulation、Transfer Learning
- 分类：研究旁支
- 暂存原因：需要先掌握模型偏差、域随机化与评价协议
- 重新进入条件：完成 L12、L14、L20，并有一组仿真分布外实验
