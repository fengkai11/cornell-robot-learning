# 论文笔记

本目录只记录与当前课程主题直接相关、能够帮助理解方法或实验的论文。

## 阅读优先级

1. 课程指定阅读；
2. 课程方法的经典原始论文；
3. 能解释关键失败模式或工程边界的论文；
4. 与当前 ManiSkill 实验直接相关的论文；
5. 前沿论文仅在基础模块完成后进入。

## 阅读纪律

- 不以论文数量作为进度；
- 不只摘抄摘要和结论；
- 必须区分论文声称的贡献与实验真正支持的结论；
- 必须检查基线是否公平、评价是否充分；
- 引用 SOTA 结果时记录任务、数据、指标和训练预算；
- 对 Diffusion Policy、ACT、VLA 等论文，先明确它们相对于 BC、RL 或序列建模增加了什么。
